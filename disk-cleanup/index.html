CLAUDE CODE SKILL

/disk-cleanup

Find what's eating your disk. Delete it. See the difference. Five minutes, no guesswork.

~ /disk-cleanup
also triggers on: "free up space" / "clean my disk" / "low on storage"
Install

Two ways to get it:

Download SKILL.md
save to ~/.claude/skills/disk-cleanup/
or paste this in your terminal:
~ mkdir -p ~/.claude/skills/disk-cleanup && curl -sL https://raw.githubusercontent.com/gneyal/claude-skills/main/disk-cleanup/SKILL.md -o ~/.claude/skills/disk-cleanup/SKILL.md

Restart Claude Code. Type /disk-cleanup and go.
